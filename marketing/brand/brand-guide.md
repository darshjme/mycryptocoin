## 1. Logo

### Primary Logo
The MyCryptoCoin logo is a wordmark: **"MyCryptoCoin"** set in **Manrope ExtraBold (800 weight)**.

The word "Crypto" within the wordmark features a gradient fill from **Primary Blue (#3b82f6)** to **Primary Purple (#8b5cf6)** at a 135-degree angle. The remaining characters ("My" and "Coin") are set in the primary text color appropriate for the background.

### Logo Variants

| Variant | Description | Use Case |
|---------|-------------|----------|
| **Full Wordmark (Dark BG)** | "My" in white, "Crypto" in gradient, "Coin" in white, on dark background | Website, pitch decks, dark UI |
| **Full Wordmark (Light BG)** | "My" in dark (#0a0a0f), "Crypto" in gradient, "Coin" in dark, on white/light background | Print materials, light contexts |
| **Monochrome White** | Entire wordmark in white, no gradient | Single-color applications on dark |
| **Monochrome Dark** | Entire wordmark in #0a0a0f, no gradient | Single-color applications on light |
| **Abbreviated (MC)** | "MC" in Manrope ExtraBold within a rounded square with gradient background | Favicons, app icons, small sizes |

### Logo Sizing
- **Minimum width:** 120px (digital), 30mm (print)
- **Clear space:** Maintain padding equal to the height of the letter "M" on all sides
- **Favicon/App icon:** Use the abbreviated "MC" mark at 32x32, 180x180, and 512x512

### Logo Usage Rules

**Do:**
- Use the logo on backgrounds with sufficient contrast
- Maintain the correct gradient direction (135 degrees, blue to purple)
- Use the provided logo files without modification
- Ensure the clear space is respected at all times

**Don't:**
- Stretch, skew, or distort the logo
- Change the gradient colors or direction
- Add drop shadows, outlines, or effects to the logo
- Place the logo on busy or low-contrast backgrounds
- Recreate the logo in a different typeface
- Rotate the logo
- Separate "My", "Crypto", and "Coin" onto different lines

## 3. Typography

### Typeface
**Manrope** is our sole typeface. It is a modern, geometric sans-serif that conveys professionalism and technical precision while remaining approachable.

Source: [Google Fonts — Manrope](https://fonts.google.com/specimen/Manrope)

### Type Scale

| Element | Size | Weight | Line Height | Letter Spacing |
|---------|------|--------|-------------|----------------|
| **Display Heading** | 3.5rem (56px) | ExtraBold (800) | 1.1 | -0.03em |
| **H1** | 2.75rem (44px) | ExtraBold (800) | 1.15 | -0.02em |
| **H2** | 2rem (32px) | Bold (700) | 1.2 | -0.02em |
| **H3** | 1.5rem (24px) | Bold (700) | 1.3 | -0.01em |
| **H4** | 1.15rem (18.4px) | Bold (700) | 1.3 | 0 |
| **Body Large** | 1.125rem (18px) | Regular (400) | 1.7 | 0 |
| **Body** | 1rem (16px) | Regular (400) | 1.6 | 0 |
| **Body Small** | 0.875rem (14px) | Medium (500) | 1.5 | 0 |
| **Caption** | 0.75rem (12px) | Medium (500) | 1.4 | 0 |
| **Label** | 0.75rem (12px) | Bold (700) | 1 | 0.15em |
| **Code** | 0.875rem (14px) | Regular (monospace fallback) | 1.6 | 0 |

### Font Weights Used
- **400 (Regular):** Body text, descriptions
- **500 (Medium):** Small text, metadata, UI labels
- **600 (SemiBold):** Subheadings, emphasized body text
- **700 (Bold):** Section headings, card titles, buttons
- **800 (ExtraBold):** Hero headings, display text, stats, logo

### Typography Rules

**Do:**
- Use Manrope for all brand communications
- Maintain the specified line heights for readability
- Use the gradient text effect sparingly (headings only)
- Use negative letter spacing for large headings

**Don't:**
- Use more than 3 font weights on a single page/section
- Set body text below 14px
- Apply gradient text to body copy or small text
- Use ALL CAPS except for labels and category tags

## 4. Voice & Tone

### Brand Voice Attributes

**Professional**
We are a financial technology company. Our communications are clear, accurate, and trustworthy. We don't use hype or make promises we can't keep.

**Developer-Friendly**
We respect our technical audience. We provide precise documentation, real code examples, and straightforward API design. We don't dumb things down, but we don't gatekeep either.

**Trustworthy**
We handle money. Trust is everything. We are transparent about fees, security, and limitations. We never hide costs or use deceptive language.

**Global**
We serve businesses worldwide. Our language is inclusive, culturally aware, and avoids region-specific idioms or references that exclude international audiences.

**Concise**
We say what we mean in as few words as possible. We prefer short sentences, clear structure, and scannable content.

### Writing Guidelines

| Context | Tone | Example |
|---------|------|---------|
| Marketing copy | Confident, clear, benefit-focused | "Accept 10+ cryptos with one integration." |
| Documentation | Precise, instructive, neutral | "Send a POST request to /api/v1/payments with the following parameters." |
| Error messages | Helpful, direct, solution-oriented | "API key not found. Check that your key is correct and not expired." |
| Email | Warm, professional, action-oriented | "Your payment of 0.015 ETH has been confirmed." |
| Social media | Sharp, engaging, value-driven | "Most processors charge 1-3%. We charge 0.5%. The math is simple." |

### Key Phrases

**Use:**
- "The Stripe of Crypto Payments"
- "Accept crypto in 5 minutes"
- "0.5% per transaction"
- "Instant settlement"
- "Enterprise-grade security"
- "One API, every crypto"

**Avoid:**
- "To the moon" or any speculative crypto slang
- "Blockchain revolution" or hyperbolic claims
- "Decentralized" (we are a centralized service, be honest)
- "Web3" as a buzzword without substance
- "Trust us" (show, don't tell)
- Excessive exclamation marks

## 5. Imagery & Iconography

### Style
- Clean, minimal, geometric
- Dark backgrounds with subtle gradient accents
- UI mockups and code snippets over stock photography
- Abstract data visualizations over literal crypto imagery (avoid coin/blockchain cliches)

### Iconography
- Line icons with 1.5-2px stroke weight
- Rounded corners consistent with border-radius: 12-16px
- Icons should be single-color (use brand colors)
- Prefer functional icons over decorative ones

### Photography
- If photography is used, prefer:
  - Developer workspaces
  - Clean, modern office environments
  - Global/diverse representation
  - Dark, moody lighting consistent with brand
- Avoid:
  - Gold bitcoin coins
  - Stock photos of people pointing at screens
  - Laser-eye memes or crypto bro imagery
  - Overly futuristic/sci-fi visuals

## 6. Component Styling

### Buttons
- **Primary:** Gradient background (Primary Gradient), white text, Manrope Bold 700, border-radius 12px, padding 14px 32px
- **Secondary:** Card BG background, card border, white text, Manrope Bold 700, same radius and padding
- **Hover state:** Subtle brightness increase (filter: brightness(1.1))

### Cards
- Background: rgba(255, 255, 255, 0.05)
- Border: 1px solid rgba(255, 255, 255, 0.08)
- Border-radius: 16px
- Padding: 32px
- Hover: slight translateY(-2px) with border brightening

### Input Fields
- Background: rgba(255, 255, 255, 0.05)
- Border: 1px solid rgba(255, 255, 255, 0.1)
- Border-radius: 10px
- Padding: 12px 16px
- Focus: border-color transitions to Primary Blue

### Badges / Tags
- Background: color at 15% opacity (e.g., rgba(6, 182, 212, 0.15))
- Text: full color value
- Border-radius: 100px (pill shape)
- Padding: 6px 16px
- Font: Manrope Bold 700, 0.75rem, uppercase, 0.1em letter-spacing

## 7. Do's and Don'ts Summary

### Do's
- Always use the Manrope typeface
- Maintain the dark theme as the primary brand expression
- Use gradients for emphasis and hierarchy, not decoration
- Keep messaging benefit-driven and specific (numbers > adjectives)
- Prioritize clarity over cleverness
- Test all visuals on both dark and light backgrounds
- Use consistent border-radius values (10px for small, 12px for medium, 16px for large)
- Credit the brand correctly: "MyCryptoCoin" (one word, capital M, C, C)

### Don'ts
- Don't use light/white theme as the primary brand expression
- Don't apply gradients to large background areas (use as accents only)
- Don't use more than 2 gradient colors in a single element
- Don't write "My Crypto Coin" (three words) or "Mycryptocoin" (lowercase)
- Don't use Comic Sans, Papyrus, or any typeface other than Manrope
- Don't make unsubstantiated claims ("the best", "number one")
- Don't use cryptocurrency price predictions or financial advice in marketing
- Don't use competitor names in paid advertising copy
- Don't place colored text on colored backgrounds without checking contrast (minimum WCAG AA: 4.5:1 for body, 3:1 for large text)
- Don't use the brand to endorse, promote, or associate with specific crypto tokens, NFTs, or DeFi protocols
